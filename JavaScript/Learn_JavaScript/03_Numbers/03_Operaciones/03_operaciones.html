Operaciones

Como puedes esperar, los números pueden multiplicarse y dividirse. Ya hemos hecho un ejemplo de multiplicación al principio del curso, usando el operador *. Para la división, debes usar el operador /.

División con resto

También puedes usar el operador de resto % que devuelve el resto de la división. Aquí tienes un ejemplo:

console.log(8 % 2); // El resto de la división es 0

console.log(7 % 2); // El resto de la división es 1

Prueba

Aquí se muestra cómo se calcula el resto de la división de 8 % 2:

8 / 2 = 4

4 / 2 = 2

=> el resto de la división es 0 porque 8 = 4 * 2 + 0

Mientras que para 7 % 2:

Empezamos dividiendo 7 por 2: 7 / 2 = 3.5. Luego, tomamos el número entero de este número y eliminamos el decimal: 3.

De otra manera, puedes escribir 7 como i3 * 2 + 1. Ese 1 al final es el resto de la división.

Usaremos el resto de la división en 2 capítulos a partir de ahora para verificar si un número es par o impar.

Resto (%)

El operador resto (%) devuelve el resto de la división entre dos operandos. Siempre toma el signo del dividendo.

Prueba

console.log(13 % 5);

// Resultado esperado: 3

 

console.log(-13 % 5);

// Resultado esperado: -3

 

console.log(4 % 2);

// Resultado esperado: 0

 

console.log(-4 % 2);

// Resultado esperado: -0

Prueba

Tener en cuenta que en la mayoría de los lenguajes de programación, el operador de resto es '%', pero en otros (como Python, Perl) es el operador módulo. Ambos son iguales para valores positivos, pero cuando el divisor y el dividendo son de signos distintos, se obtienen distintos resultados. Para obtener el módulo en JavaScript, en lugar de a % n, usa ((a % n ) + n ) % n.

Sintaxis

Operador: var1 % var2

Ejemplos

Resto con dividendo positivo

console.log(12 % 5); // 2

console.log(1 % -2); // 1

console.log(1 % 2); // 1

console.log(2 % 3); // 2

console.log(5.5 % 2); // 1.5

Prueba

Resto con dividendo negativo

console.log(-12 % 5); // -2

console.log(-1 % 2); // -1

console.log(-4 % 2); // -0

Prueba

Resto con dividendo NaN

console.log(NaN % 2); // NaN

Prueba

Resto con Infinity

console.log(Infinity % 2); // NaN

console.log(Infinity % 0); // NaN

console.log(Infinity % Infinity); // NaN

Métodos de números

Si bien hay otros métodos que puedes llamar en números, no se usan muy comúnmente. Lo que se usa comúnmente, sin embargo, es el objeto Math, que contiene métodos como min(), max(), round(), etc.

Aquí hay algunos ejemplos:

console.log(Math.round(2.1)); // 2

console.log(Math.round(2.6)); // 3

console.log(Math.round(2.5)); // 3

 

console.log(Math.floor(2.1)); // 2

console.log(Math.floor(2.6)); // 2

console.log(Math.floor(2.5)); // 2

 

console.log(Math.ceil(2.1)); // 3

console.log(Math.ceil(2.6)); // 3

console.log(Math.ceil(2.5)); // 3

Prueba

Resumen

  • El operador de resto (%) devuelve el resto de la división entre dos números.
