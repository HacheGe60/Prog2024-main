Conversión de cadena a número

En algunos escenarios (explicados a continuación), querrás convertir de una cadena a un número. Para eso, tendrás que usar el método Number.parseInt().

Ejemplo:

let str = "42";

Number.parseInt(str, 10); // 42

Hay mucho sucediendo aquí, así que vamos a desglosarlo paso a paso.

La función se llama Number.parseInt(). Sí, incluyendo la parte Number.. Esto se debe a que hay un objeto global llamado Number.que contiene un método llamado parseInt().

El método Number.parseInt() espera 2 parámetros:

Number.parseInt(string, radix);

El primer parámetro es la cadena que deseas convertir en un número. El segundo argumento es el radix que se utilizará en la conversión.

El radix es la base del sistema numérico que deseas usar. Para la mayoría de los casos de uso, el radix que querrás usar es 10, que representa la forma en que contamos números en nuestra vida cotidiana. Este sistema se llama sistema decimal (porque tenemos 10 dedos, por lo que usamos los dígitos del 0 al 9).

Otro ejemplo de radix es 2, que representa binario (un sistema numérico utilizado por las computadoras). Si deseas profundizar más en este concepto, consulta esta página simplificada de Wikipedia sobre bases matemáticas.

Wikipedia Base (aritmética)

En un sistema de numeración posicional, se le llama base al número que define el orden de magnitud en que se ve incrementada cada una de las cifras sucesivas que componen el número. Es también la cantidad de símbolos presentes en dicho sistema. Por ejemplo, el sistema de numeración decimal (el más utilizado en la actualidad) utiliza como base el número 10 (diez): hay 10 símbolos o dígitos, y cada uno de ellos se incrementa en un orden de magnitud de 10 por cada posición consecutiva.

En cualquier sistema de numeración, el número x y su base y se denotan convencionalmente como (x)y, esto se cumple para todos los sistemas salvo el decimal, por ser la manera más habitual de expresar valores, donde se omite la base. Así, por ejemplo,(100)10 (o sin la base) es el número 100 en el sistema decimal; y (100)2 es el número 4 en el sistema binario.

Como resumen rápido, el radix generalmente será 10. Si no estás seguro de qué radix elegir, lo más probable es que sea 10.

¿Se puede obviar el radix?

Aunque el radix es un parámetro opcional, no debes omitirlo. Eso es porque no siempre se establece como 10 por defecto.

Así que asegúrate de siempre pasar el radix como segundo parámetro.

Number.parseInt(string, radix) no siempre se establece como un radix de 10 de forma predeterminada.

Si intentas usar Number.parseInt() sin un radix de 10, funcionará. Sin embargo, hay algunos casos límite (números que comienzan con 0x) que se romperían. Por lo tanto, para estar seguro, siempre se recomienda pasar el radix.

Asegúrate de especificar siempre el radix para evitar sorpresas desagradables.

Number.parseInt()

El método estático Number.parseInt() analiza un argumento de cadena y devuelve un número entero de la raíz o base especificada.

Prueba

function roughScale(x, base) {

  const parsed = Number.parseInt(x, base);

  if (Number.isNaN(parsed)) {

    return 0;

  }

  return parsed * 100;

}

 

console.log(roughScale(" 0xF", 16));

// resultado esperado: 1500

 

console.log(roughScale("321", 2));

// resultado esperado: 0

Prueba

Sintaxis

Number.parseInt(string)

Number.parseInt(string, radix)

Parámetros

string

El valor a analizar, forzado a una cadena. Se ignoran los espacios en blanco iniciales en este argumento.

radix (opcional)

Un número entero entre 2 y 36 que representa la base (la base en los sistemas numéricos matemáticos) del string.

Si radix no está definida o es 0, se supone que es 10, excepto cuando el número comienza con los pares de unidades de código 0x o 0X, en cuyo caso se asume una base de 16.

Valor devuelto

Un entero analizado a partir del string dado.

Si radix es menor que 2 o mayor que 36, o el primer carácter que no es un espacio en blanco no se puede convertir en un número, se devuelve NaN.

Ejemplos

Number.parseInt vs. parseInt

Este método tiene la misma funcionalidad que la función global parseInt():

Number.parseInt === parseInt; // true

Su propósito es la modularización de globales. Consulte parseInt() para obtener más detalles y ejemplos.

Notas de legado

JavaScript es un lenguaje en evolución que tiene más de 25 años. Sigue cambiando y evolucionando. Las notas de legado explicarán algunos comportamientos confusos o funciones antiguas que quizás hayas encontrado hace un tiempo.

Si estás aprendiendo JavaScript por primera vez, entonces no necesitas dedicar mucho tiempo a esas notas de legado.

Number.parseInt() y parseInt() son exactamente el mismo método. Se prefiere el uso de Number.parseInt() sobre parseInt().

Hace un tiempo, parseInt(string, radix) era la única forma de convertir números, sin embargo, tiempo después, esta función ha sido clonada bajo el objeto Number y se convirtió en Number.parseInt(string, radix) en un esfuerzo por agrupar funciones similares juntas bajo su objeto relevante.

Ambas funcionan exactamente igual. Recomendamos que te adhieras al enfoque moderno, que es Number.parseInt().

Casos de uso para convertir una cadena a un número

Hay varias razones por las que querrás convertir una cadena a un número, pero la más común es cuando el número es ingresado por el usuario en un cuadro de texto o el número se está leyendo desde el DOM (lo que se explica más adelante).

Como verás en el próximo desafío, estos valores siempre serán una cadena (incluso si el usuario escribe un número). Por lo tanto, es tu trabajo convertirlo en un número.

Si olvidas convertir una cadena a un número, verás que la suma prevista se comporta como una concatenación.

let a = 10;

let b = '20'; // nos olvidamos de convertir esta cadena a un número

console.log(a + b); // '1020' (concatenación en vez de suma)

Prueba

El método Number.parseInt() intentará convertir la cadena que recibe en un número. Como puedes ver a continuación, funciona con más frecuencia cuando la cadena comienza con un número y termina con valores no numéricos:

console.log(Number.parseInt('123abc', 10)); // 123

console.log(Number.parseInt('5 meters', 10)); // 5

Prueba

Resumen

  • Convertir de cadena a número: Number.parseInt(value, 10).
  • Number.parseInt() es el nombre de la función que estás llamando.
  • Debes especificar el radix 10.
  • Asegúrate de especificar siempre el radix para evitar sorpresas desagradables.

Prácticas

Obtener la siguiente edad I

Completa la función getNextAge para que devuelva la edad del próximo año (sumando 1 a la edad actual).

Ten en cuenta que la age la escribe el usuario en un cuadro de texto (podés probar en la pestaña "Navegador"

No te preocupes por obtener NaN cuando el cuadro de texto esté vacío por ahora, lo trataremos en un capítulo futuro.

export function getNextAge(age) {

  return Number.parseInt(age) + 1;

}

Obtener el ancho de la caja

Completa la función getBoxWidth para que devuelva el ancho (como número) de la cadena de texto que recibes.

Ten en cuenta que la cadena que recibe se parece a 50px o 100px (que se usa en CSS), mientras que las funciones deberían devolver un número a partir de estas cadenas.

  return Number.parseInt(value);
