Acceso a caracteres

Puedes acceder a un carácter específico en una cadena usando la sintaxis de corchetes [ ].

Tienes que proporcionar el índice del carácter al que deseas acceder, comenzando desde 0.

Tomemos un ejemplo donde la variable language tiene el valor: "JavaScript". Aquí está cómo puedes acceder al primer carácter, al segundo y al tercero:

const language = "JavaScript";

 

language[0]; // primer carácter

language[1]; // segundo carácter

language[2]; // tercer carácter

Prueba

Si deseas depurar tu código y ver el resultado de language[1] en la consola, debes registrarlo de la siguiente manera:

console.log(language[1]);

Prueba

Combinándolo con la longitud

También puedes combinar el acceso a caracteres con la propiedad .length. Entonces, usando la misma variable language, aquí tienes cómo obtener el segundo carácter desde el final:

const language = "JavaScript";

 

language[language.length - 2]; // "p" porque es el segundo carácter del final de "JavaScript"

Prueba

Ten en cuenta que language[language.length] será undefined porque el acceso a caracteres comienza en 0.

De esta manera, para una cadena de 9 caracteres, 8 es la posición del último carácter.

El método .at(index)

Desde 2022, JavaScript ahora tiene un método .at() que lee el carácter en un determinado índice, que también puede ser negativo.

Veamos algunos ejemplos:

const language = "JavaScript";

console.log(language.at(0)); // "J"

console.log(language.at(1)); // "a"

console.log(language.at(-1)); // "t"

console.log(language.at(-2)); // "p"

Prueba

El principal caso de uso para el método .at() son los índices negativos, lo que lo hace más fácil que depender de la propiedad .length. Sin embargo, puedes usar el enfoque que prefieras.

Cuando se especifica un índice negativo, se comienza a contar desde el final. -1 es el último carácter, -2 es el anterior, etc.

Puedes seguir usando la sintaxis de corchetes para todos los demás casos de uso, sabiendo que language[-1] devuelve undefined. Por lo tanto, siempre que desees usar un índice negativo, debes usar el método .at().

Resumen

  • Los corchetes [índice] se utilizan para acceder a un índice específico de una cadena.
  • El índice comienza en 0. Por lo tanto, el primer carácter es el índice 0.
  • Puedes combinarlo con la longitud de una cadena para obtener otro carácter en otra posición.
  • El método .at() te permite leer un carácter en un índice determinado (que también puede ser negativo).

Prácticas

Obtener el primer carácter

Completa la función getFirstCharacter para que devuelva el primer carácter del parámetro name que recibe.

function getFirstCharacter(name) {

    return name[0];

}

 

// Ejemplos

console.log(getFirstCharacter("Amsterdam")); // A

console.log(getFirstCharacter("Japan")); // J

Prueba

Obtener el último carácter

Completa la función getLastCharacter para que devuelva el último carácter del parámetro name que recibe.

function getLastCharacter(name) {

    return name.at(-1);

}

 

// Ejemplos

console.log(getLastCharacter("Sam")); // m

console.log(getLastCharacter("Alex")); // x

console.log(getLastCharacter("Cherley")); // y
