Strings

En JavaScript, podés hacer cadenas de texto usando comillas dobles o comillas simples.

Por ejemplo:

"Esta es una cadena de texto";

'¡Esta es otra cadena de texto!'

No importa si usás comillas dobles o comillas simples, es lo mismo. Ninguna de las dos formas te permite poner variables dentro de la cadena. Eso lo veremos más adelante, en otra lección.

Propiedad de cadena

La propiedad .length se usa para devolver la longitud de la cadena.

Acá tenés un ejemplo de cómo saber cuántos caracteres tiene "Linda!":

"Linda!".length; // 6

Prueba

Si tenés una variable que se llama text, así es como podés saber cuántas letras tiene:

let text = "Hola Mundo"

text.length; // 10

Prueba

Métodos básicos de cadenas

Aquí hay algunos métodos comunes que puedes llamar en cadenas:

.toLowerCase()

Esto devolverá una nueva cadena que tiene todos sus caracteres en minúscula.

"HOLA MUNDO".toLowerCase() // "hola mundo"

Prueba

.length no necesita paréntesis porque es como una característica de la cadena, mientras que .toLowerCase() sí necesita paréntesis porque es algo que le estás pidiendo que haga a la cadena.

.toUpperCase()

Esto devolverá una nueva cadena que tiene todos sus caracteres en mayúscula.

"hola mundo".toUpperCase() // "HOLA MUNDO"

Prueba

Visualizar una variable (o expresión)

Cuando resuelvas desafíos, no dudes en usar console.log() para visualizar una variable o expresión. Aparecerá en la pestaña "consola".

function sum(a, b) {

    console.log(a);

    console.log(a + b);

    return a + b;

}

 

// Ejemplo

sum(1, 3);

Prueba

Con el código anterior, verás 1 y luego 4 registrado en la consola debido a console.log(a) y console.log(a + b).

Ten en cuenta que console.log() no es un reemplazo para return. Todavía tienes que usar return después de console.log().

Resumen

  • Se pueden crear cadenas de texto (strings) con " o '.
  • .length es una propiedad que devuelve la longitud de una cadena.
  • .toLowerCase() y .toUpperCase() convierten una cadena en minúscula o mayúscula.
  • Los paréntesis () son obligatorios en las llamadas a funciones. .length es una propiedad que ya está precalculada; por lo tanto, no necesita paréntesis.
  • console.log(...) se usa para depurar y NO es un reemplazo para return.

Prácticas

Longitud de la cadena

Completa la función getCharCount para que devuelva la cantidad de caracteres en el parámetro str que recibe.

function getCharCount(str) {

    return str.length;

}

 

// Ejemplos

alert(getCharCount("Sam")); // 3

alert(getCharCount("Alex 123")); // 8

alert(getCharCount("Charley is here")); // 15

Prueba

Grita mi nombre

Completa la función shoutMyName para que devuelva el parámetro name que recibe, todo en mayúscula.

function shoutMyName(name) {

    return name.toUpperCase();

}

 

// Ejemplos

alert(shoutMyName("Sam")); // SAM

alert(shoutMyName("Charley")); // CHARLEY

alert(shoutMyName("alex")); // ALEX

Prueba

Cadena en minúsculas

Completa la función lowerName para que devuelva el parámetro name que recibe en minúscula.

function lowerName(name) {

    return name.toLowerCase();

}

 

// Ejemplos

alert(lowerName("Sam")); // sam

alert(lowerName("ALEX")); // alex
