Literales de plantilla

Ya sabes que puedes crear cadenas con comillas dobles o comillas simples, pero como ya sabes, estas cadenas no admiten interpolación.

Las cadenas de plantilla, sin embargo, admiten interpolación y otras características interesantes.

Tu primer literal de plantilla

`Este es un literal de plantilla`

La única diferencia es que las cadenas de plantilla comienzan y terminan con un acento grave `.

Cadenas de texto multilíneas

A diferencia de las cadenas de comillas simples y dobles, las cadenas de plantilla pueden abarcar varias líneas. Aquí hay un ejemplo:

let text = `Esta es una cadena

de texto

con varias

líneas

`;

 

console.log(text);

Prueba

Esto no hubiera sido posible con una cadena normal (comillas simples o dobles).

Interpolación

Las cadenas de plantilla admiten interpolación. Esto significa que puedes escribir una variable en tu cadena y obtener su valor. La sintaxis es sencilla, envuelves el nombre de tu variable con un signo de dólar y llaves. Tomemos un ejemplo donde tenemos una variable language con un valor de "JavaScript".  

let language = "JavaScript";

console.log(`Mi lenguaje favorito es ${language}`); // "Mi lenguaje favorito es JavaScript"

Prueba

Recuerda que la interpolación de cadenas solo funciona con comillas invertidas. Si alguna vez lo intentas y no funciona, verifica que estés usando comillas invertidas en lugar de comillas simples o dobles.  

Ejemplos adicionales

Cómo escribir una cadena de varias líneas

const text = `Usando el carácter de comilla invertida,

puedes definir una cadena que

abarque varias líneas.`

console.log(text);

Prueba

Como interpolar en JavaScript

const name = "Sam";

 

console.log(`Bienvenido ${name}.

Tienes ${2 * 5} notificaciones nuevas.`);

Prueba

Resumen

  • Una cadena de plantilla es una cadena creada con el carácter de comilla invertida: `.
  • Las cadenas de plantilla pueden abarcar varias líneas.
  • Las cadenas de plantilla admiten interpolación con la sintaxis ${nombreVariable}.

Prácticas

Interpolación de cadenas

Completá la función sayHello de manera que interpole la variable name en una cadena de texto 'Hola name'.

function sayHello(name) {

  return `Hola ${name}`;

}

 

// Ejemplos

console.log(sayHello("Alex")); // "Hola Alex"

console.log(sayHello("Sam")); // "Hola Sam"

Prueba

Interpolación de cadenas avanzada

Completa la función getFullName de modo que devuelva el nombre completo de la persona utilizando interpolación.

function getFullName(firstName, lastName) {

  return `${firstName} ${lastName}`;

}

 

// Ejemplos

console.log(getFullName("Sam", "Doe")); // "Sam Doe"

console.log(getFullName("Alex", "Blue")); // "Alex Blue"

Es muy común en JavaScript devolver una cadena que representa algún código HTML. De hecho, haremos esto con bastante frecuencia en los capítulos de DOM. La interpolación de cadenas resulta útil aquí porque admite cadenas de varias líneas e interpolación.

<tr>

donde label here y value here son reemplazados por los parámetros label y value respectivamente. Asegúrate de mirar el código en index.js para ver cómo se está utilizando esta función (es probable que el código DOM no sea comprensible en este momento, ya que aún no lo hemos explicado).

export function renderTableRow(label, value) {

  console.log(label, value);

  return `<tr>

            <td>${label}</td>

            <td>${value}</td>

          <tr>`;

}

Capitalizar palabra

Completa la función capitalize para que capitalice el parámetro word que recibe. No existe un método capitalize en JavaScript, por lo que debes escribirlo tú mismo.

Mira el ejemplo de uso para entender lo que significa capitalizar, luego mira las pistas si aún no está claro.

function capitalize(word) {

  return word.at(0).toUpperCase() + word.substring(1).toLowerCase();

}

 

// Ejemplos

console.log(capitalize("sam")); // "Sam"

console.log(capitalize("ALEX"));// "Alex"

console.log(capitalize("chARLie"));// "Charlie"

Proyecto Variaciones de nombres

En este proyecto, ¡practicarás las habilidades de cadenas que acabas de adquirir!

Completa las funciones dadas como se describe en los comentarios y prueba la aplicación que acabas de construir en la pestaña "NAVEGADOR".

Tómate tu tiempo, comete la menor cantidad de errores que puedas y aprende de ellos.

export function getNumberOfChars(name) {

  // número de caracteres en la variable name

  return name.length;

}

 

export function getFirstChar(name) {

  // primer carácter de la variable name

  return name.at(0);

}

 

export function getLastChar(name) {

  // último carácter de la variable name

  return name.at(-1);

}

 

export function getLower(name) {

  // variable name en minúsculas

  return name.toLowerCase();

}

 

export function getUpper(name) {

  // variable name en mayúsculas

  return name.toUpperCase();

}

 

export function getCapitalized(name) {

  // variable name capitalizada

  return name.at(0).toUpperCase() + name.substring(1).toLowerCase();

}

 

Prueba

Resumen del capítulo

  • Puedes crear cadenas de texto con " o '.
  • .length es una propiedad que te indica la longitud de una cadena.
  • .toUpperCase() es una función que convierte la cadena a mayúsculas.
  • .toLowerCase() es una función que convierte la cadena a minúsculas.
  • Los paréntesis () son obligatorios en las llamadas a funciones. .length es una propiedad precalculada, por lo tanto, no necesita paréntesis.
  • console.log(...) se usa para depurar y no es un reemplazo para return.
  • Los corchetes [índice] se usan para acceder a un índice específico de una cadena.
  • El índice comienza en 0. Por lo tanto, el primer carácter es el índice 0.
  • Puedes combinarlo con la longitud de una cadena para obtener otro carácter en otra posición.
  • El método .at() te permite leer un carácter en un índice determinado (que también puede ser negativo).
  • Una subcadena es una parte o una porción de una cadena.
  • string.substring(indexStart, indexEnd) se usa para devolver una porción de la cadena.
  • indexStart: la posición del primer carácter que deseas incluir.
  • indexEnd: la posición del primer carácter que deseas ignorar.
  • El argumento indexEnd es opcional, lo que significa que puedes omitirlo.
  • El operador + es usado para sumar 2 números.
  • El operador + es usado para concatenar dos cadenas.
  • Una cadena de plantilla es una cadena creada con el carácter de comilla invertida: `.
  • Las cadenas de plantilla pueden extenderse a varias líneas.
  • Las cadenas de plantilla admiten interpolación con la sintaxis ${variableName}.
