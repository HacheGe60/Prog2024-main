Substrings

Una subcadena es una parte o una porción de una cadena. Por ejemplo, "rain" es una subcadena de la cadena "brain" porque puedes obtener "rain" tomando los últimos 4 caracteres.

A menudo, cuando trabajas con cadenas, necesitas obtener algunos caracteres de una cadena en lugar de todos. Por lo tanto, utilizamos el método substring.

Sintaxis de substring

La sintaxis de una función te brinda una explicación de los parámetros que debes pasar para ese método. Veamos la sintaxis de substring:

unaCadena.substring(indiceInicial[, indiceFinal])

Esto significa que cuando llamas a substring, puedes pasarle 2 parámetros: el primero para indiceInicial y el segundo para indiceFinal.

  • indiceInicial: la posición del primer carácter que uno quiere incluir.
  • indiceFinal: la ubicación del último carácter que uno quiere ignorar.

Esto significa que un indiceFinal de 5 solo incluirá hasta el carácter en la posición 4.

La combinación de estos 2 te dará una subcadena.

Tomemos un ejemplo con una variable llamada language con un valor "JavaScript", y obtengamos la subcadena con indiceInicial de 1 e indiceFinal de 4. Esto devolverá una cadena compuesta por todos los caracteres de las posiciones 1 a 3, ya que 4 es el primer carácter que se ignora.

El resultado de esta subcadena es "ava".

Esta es la forma de hacerlo en JavaScript:

const language = "JavaScript";

console.log(language.substring(1, 4));// "ava"

Parámetros opcionales

El parámetro indiceFinal es opcional, lo que significa que puedes pasar el indiceInicial y asumirá que el indiceFinal es igual a la longitud de la cadena. Aquí hay un ejemplo:

const language = "JavaScript";

console.log(language.substring(4)); // "Script"

Nota de la vieja escuela

Si ya sabes un poco de JavaScript, es posible que hayas usado otro método que realiza un resultado similar. Puedes encontrar el nombre de la función a continuación, pero no recomendamos que lo uses porque está obsoleto.

No uses el método .substr, ya que está obsoleto y funciona de manera diferente. Siempre usa el método .substring.

Resumen

  • Una subcadena es una parte o una porción de una cadena.
  • unaCadena.substring(indiceInicial[, indiceFinal]) se usa para devolver una porción de la cadena.
  • indiceInicial indica la posición del primer carácter que uno quiere incluir.
  • indiceFinal indica la ubicación del último carácter que uno quiere ignorar.
  • el indiceFinal es opcional, lo que significa que puedes pasar el indiceInicial directamente.

Prácticas

Ignora el primer carácter

Completa la función skipFirstCharacter para que devuelva todos los caracteres excepto el primero del parámetro text que recibe.

function skipFirstCharacter(text) {

  return text.substring(1);

}

 

// Ejemplos

console.log(skipFirstCharacter("Xcode")); // code

console.log(skipFirstCharacter("Hello")); // ello
