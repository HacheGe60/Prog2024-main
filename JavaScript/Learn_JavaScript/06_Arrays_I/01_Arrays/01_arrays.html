Arrays

Los arrays en JavaScript te permiten almacenar múltiples elementos en la misma variable. Puedes almacenar números, cadenas, booleanos, arrays, objetos y más. Estos pueden mezclarse dentro del mismo array.

Aquí hay algunos ejemplos:

const users = []; // array vacío

const grades = [10 // array de números

const attendees = ['Sam', 'Alex']; // array de cadenas de texto

const values = [10, false, 'John']; // array mixto

Nombra los arrays en plural ya que pueden contener más de un elemento. Esto resultará ser especialmente útil una vez que necesitemos iterar sobre un array.

Propiedad .length

Puedes obtener el número de elementos en un array usando la propiedad .length. Por ejemplo:

[].length; // 0

 

const grades = ['10', 8, 13, 15];

grades.length; // 4

Ten en cuenta que .length es una propiedad (valor precalculado) y no una función. Por eso no debes poner () después de ella.

Array.length

La propiedad length de un objeto que es una instancia de tipo Array establece o devuelve la cantidad de elementos en esa matriz. El valor es un entero sin signo de 32 bits que siempre es numéricamente mayor que el índice más alto en la matriz.

Prueba

const clothing = ['shoes', 'shirts', 'socks', 'sweaters'];

 

console.log(clothing.length); // Resultado esperado: 4

Prueba

Descripción

El valor de la propiedad length es un número entero con un signo positivo y un valor menor que 2 a la 32a potencia (232).

let nameListA = new Array(4294967296); // 2 a la 32a potencia = 4294967296

let nameListC = new Array(-100); // signo negativo

 

console.log(nameListA.length); // RangeError: longitud de la matriz inválida

console.log(nameListC.length); // RangeError: longitud de la matriz inválida

 

let nameListB = [];

nameListB.length = Math.pow(2, 32) - 1; // establecer una longitud de la matriz menor que 2 a la 322 potencia

console.log(nameListB.length); // 4294967295

Prueba

Puedes establecer la propiedad length para truncar una matriz unidimensional en cualquier momento. Cuando extiende una matriz cambiando su propiedad length, el número de elementos reales aumenta; por ejemplo, si se establece length en 3 cuando actualmente es 2, la matriz ahora contiene 3 elementos, lo que hace que el tercer elemento sea undefined.

let arr = [1, 2, 3];

printEntries(arr);

 

arr.length = 5; // establecer la longitud de la matriz en 5 mientras que la actual es 3.

printEntries(arr);

 

function printEntries(arr) {

  let length = arr.length;

  for (let i = 0; i < length; i++) {

    console.log(arr[i]);

  }

  console.log("=== impreso ===");

}

 

// 1

// 2

// 3

// === impreso ===

// 1

// 2

// 3

// undefined

// undefined

// === impreso ===

Prueba

Pero, la propiedad length no necesariamente indica el número de valores definidos en la matriz.

Atributos de la propiedad Array.prototype.length
Sobrescribir Sí
Numerable No
Configurable No
  • Sobrescribir: si este atributo se establece en false, el valor de la propiedad no se puede cambiar.
  • Configurable: si este atributo se establece en false, cualquier intento de eliminar la propiedad o cambiar sus atributos (Sobrescribir, Configurable o Numerable) fallará.
  • Numerable: si este atributo se establece en true, la propiedad se repetirá durante los bucles for o for..in.

Ejemplos

Iterando sobre una matriz

En el siguiente ejemplo, la matriz numbers se itera a través de la propiedad length. El valor en cada elemento se duplica.

const numbers = [1, 2, 3, 4, 5];

let length = numbers.length;

for (let i = 0; i < length; i++) {

  numbers[i] *= 2;

}

 

console.log(numbers);

Prueba

Acortando una matriz

El siguiente ejemplo acorta los numbers de la matriz a una longitud de 3 si la longitud actual es mayor que 3.

const numbers = [1, 2, 3, 4, 5];

 

if (numbers.length > 3) {

  numbers.length = 3;

}

 

console.log(numbers); // [1, 2, 3]

console.log(numbers.length); // 3

Prueba

Obtener elemento por índice

De manera similar a las cadenas, puedes obtener un elemento de un array usando la sintaxis de corchetes [] con el índice comenzando desde 0.

Por ejemplo:

const users = ['Sam', 'Alex', 'Charley'];

console.log(users[1]); // 'Alex'

Prueba

También puedes usar el método .at(index), que acepta índices negativos, lo que facilita encontrar el último elemento del array. Los índices negativos cuentan hacia atrás desde el último elemento del array. Aquí hay algunos ejemplos con .at():

const users = ['Sam', 'Alex', 'Charley', 'Blane', 'Crane'];

console.log(users.at(0)); // 'Sam'

console.log(users.at(1)); // 'Alex'

console.log(users.at(-2)); // 'Blane'

console.log(users.at(-1)); // 'Crane'

Prueba

Array.prototype.at()

El método at() recibe un valor numérico entero y devuelve el elemento en esa posición, permitiendo valores positivos y negativos. Los valores negativos contarán desde el último elemento del array.

Esto no sugiere que haya algo mal con usar la notación de corchetes. Por ejemplo, array[0] devolvería el primer elemento. Sin embargo, en lugar de usar array.length para los últimos elementos; ej. array[array.length-1] para el último elemento, puede llamar array.at(-1).

Prueba

const array1 = [5, 12, 8, 130, 44];

 

let index = 2;

 

console.log(`Un índice de ${index} da ${array1.at(index)}`);

// Resultado esperado: "Un índice de 2 da 8"

 

index =-+2

 

console.log(`Un índice de ${index} da ${array1.at(index)}`);

// Resultado esperado: "Un índice de -2 da 130"

Prueba

Sintaxis

at(indice);

Parámetros

índice

El índice (posición) del elemento del array que se devolverá. Soporta el índice relativo desde el final del array cuando se pasa un índice negativo; ej. si se usa un número negativo, el elemento devuelto se encontrará contando hacia atrás desde el final del array.

Valor de retorno

El elemento en el array que coincide con el índice dado. Devuelve undefined si el índice dado no se puede encontrar.

Ejemplos

Retorna el último valor de un array

El siguiente ejemplo proporciona una función que devuelve el último elemento encontrado en un array especificado.

// Nuestro array con objetos

const carrito = ['manzana', 'banana', 'pera'];

 

// Una función que devuelve el último elemento de un array dado

function devolverUltimo(arr) {

  return arr.at(-1);

}

 

// Obtenemos el último elemento de nuestro array 'carrito'

const objeto1 = devolverUltimo(carrito);

console.log(objeto1); // Muestra 'pera'

 

// Añade un elemento a nuestro array 'carrito'

carrito.push('naranja');

const objeto2 = devolverUltimo(carrito);

console.log(objeto2); // Muestra 'naranja'

Prueba

Comparando métodos

Este ejemplo compara diferentes maneras de seleccionar el penúltimo (uno antes del último) elemento de un Array. Mientras todos los métodos mostrados a continuación son válidos, este ejemplo resalta la concisión y la fácil lectura del método at().

// Nuestro array con elementos

const colores = ['rojo', 'verde', 'azul'];

 

// Usando la propiedad length

const lengthMetodo = colores[colores.length -2];

console.log(lengthMetodo); // Muestra 'verde'

 

// Usando el método slice(). Note que se devuelve un array

const sliceMetodo = colores.slice(-2, -1);

console.log(sliceMetodo[0]); // Muestra 'verde'

 

// Usando el método at()

const atMetodo = colores.at(-2);

console.log(atMetodo); // Muestra 'verde'

Prueba

Agregando un elemento

Se puede agregar un elemento al final de un array con el método push().

const numbers = [10, 8, 13, 15];

numbers.push(20); // devuelve 5 (la nueva longitud del array)

console.log(numbers); // [10, 8, 13, 15, 20]

Prueba

Array.push() devuelve la nueva longitud del array.

Como puedes ver, numbers.push(20) devuelve 5, que es la longitud del array. Esto a menudo es confuso para muchos desarrolladores, por lo que lo estamos destacando aquí. .push() agregará un elemento al array pero también devolverá la nueva longitud del array.

Array.prototype.push()

El método push() añade uno o más elementos al final de un array y devuelve la nueva longitud del array.

Prueba

const animales = ['cerdos', 'cabras', 'ovejas'];

 

const cantidad = animales.push('vacas');

console.log(cantidad); // ['cerdos', 'cabras', 'ovejas', 'vacas']

// Resultado esperado: 4

console.log(animales);

// Resultado esperado: Array ['cerdos', 'cabras', 'ovejas', 'vacas']

 

animales.push('pollos', 'gatos', 'perros');

console.log(animales);

// Resultado esperado: Array ['cerdos', 'cabras', 'ovejas', 'vacas', 'pollos', 'gatos', 'perros']

Prueba

Sintaxis

arr.push(element1[, ...[, elementN]]);

Parámetros

elementN

Los elementos a añadir al final del array.

Valor devuelto

La nueva propiedad length del objeto sobre el cual se efectuó la llamada.

Descripción

El método push es muy práctico para añadir valores a un array.

push es genérico intencionadamente. Este método puede ser call() o apply() a objetos que representen arrays. El método push depende de la propiedad length para decidir donde empezar a insertar los valores dados. Si el valor de la propiedad length no puede ser convertido en numérico, el índice 0 es usado. Esto permite la posibilidad de que la propiedad length sea inexistente, y en este caso length será creado.

Los únicos objetos nativos que se asemejen al array son strings objetos, aunque estos no se puedan usar en la aplicación de este método ya que son inmutables.

Ejemplos

Añadiendo elementos a un array

El siguiente código crea el array deportes que contiene dos elementos, luego añade 2 elementos más. Tras ejecutar el código, deportes contiene 4 elementos: "fútbol", "béisbol", "fútbol americano" y "natación".

const deportes = ['fútbol', 'béisbol'];

let total = deportes.push('fútbol americano', 'natación');

 

console.log(deportes); // ['fútbol', 'béisbol', 'fútbol americano', 'natación']

console.log(total); // 4

Prueba

Uniendo dos arrays

Este ejemplo usa apply() para insertar todos los elementos de un segundo array.

No uses este método si el segundo array (moreVegs en el ejemplo) es muy grande, porque el número máximo de parámetros que puede tomar una función es limitado en la práctica. Consulta apply() para obtener más detalles.

const vegetales = ['boniato', 'papa'];

const masVegs = ['apio', 'berenjena'];

 

// Combinar el segundo array en el primero

// Equivalente a vegetales.push('apio', 'berenjena');

Array.prototype.push.apply(vegetales, masVegs);

 

console.log(vegetales); // ['boniato', 'papa', 'apio', 'berenjena']

Prueba

Usando un objeto de manera similar a un array

Como se menciona anteriormente, push es intencionadamente genérico, y podemos usar eso a nuestro favor. Array.prototype.push puede funcionar bien con un objeto, como muestra este ejemplo. Ten en cuenta que no se crea un array para almacenar una colección de objetos. En su lugar, almacenamos la colección en el propio objeto y se utiliza el método call sobre Array.prototype.push para hacer creer al método que estamos tratando a un array, y simplemente funciona, gracias a la forma en que JavaScript nos permite establecer el contexto de la ejecución.

const obj = {

  length: 0,

 

  addElem: function addElem(elem) {

  // obj.length is automatically incremented

  // every time an element is added.

  [].push.call(this, elem);

  }

};

 

obj.addElem({});

obj.addElem({});

console.log(obj.length); // 2

Prueba

Tenga en cuenta que aunque obj no es un array, el método push ha incrementado satisfactoriamente la propiedad length de obj tal y como si se tratara de un array.

Arrays & const

Aunque la variable numbers = []; // comienza con un array vacío se definió con const, pudimos insertar nuevos datos en ella.

Eso se debe a que const significa que solo puedes asignar la variable una vez cuando se define. Pero no significa que la variable sea inmutable. Su contenido puede cambiar.

¿Cuál es el beneficio de declararla como const? El beneficio es que una vez que la defines como un array, siempre permanecerá como un array, lo que significa que puedes llamar con seguridad a los métodos del array en ella. Sin embargo, el contenido del array puede cambiar.

const numbers = []; // comienza con un array vacío

numbers.push(10); // devuelve 1 (nueva longitud del array)

console.log(numbers); // [10] sigue siendo un array pero cambió su contenido

numbers.push(20); // devuelve 2 (nueva longitud del array)

console.log(numbers); // [10, 20] sigue siendo un array pero cambió su contenido

Resumen

  • const data = [1, 2, 3] es un array que contiene 3 números.
  • array.length devuelve el número de elementos dentro del array.
  • array.push(x) te permite agregar la variable x al final del array. También devuelve la nueva longitud del array (después de que se haya realizado el push).
  • Los arrays definidos con const no son constantes porque puedes cambiar los elementos dentro de ellos. Sin embargo, no puedes reasignarlos a otro valor, por lo que siempre serán un array.

Añadir una calculadora a las aplicaciones

Usar cualquier app
