Funciones Básicas

Comencemos con funciones básicas.

function suma(x, y) {

    return x + y;

}

Este fragmento de código define una función llamada suma.

Ahora podés usar la función suma(1, 3), que te va a dar 4.

Podés usar la función otra vez con otros números, por ejemplo suma(2, 5), y te va a dar el resultado de 2 + 5, que es 7.

Devolviendo el resultado.

En JavaScript, tenés que decir qué valor devuelve la función. Si te olvidás de poner return, la función va a devolver undefined.

Si usás return la función se termina.

function suma(x, y) {

    return x + y;

    console.log("Hola Mundo"); // esto NUNCA correrá.

}

Si usás return, la función se va a terminar y va a devolver el resultado (x + y), así que lo que venga después no se va a ejecutar.

Prácticas

Suma básica

Completá la función suma para que devuelva la suma de a y b.

function suma(a, b) {

    return a + b;

}

alert(suma(1, 3)); // 4

alert(suma(2, 5)); // 7
