Parámetros por defecto

Antes de aprender sobre las funciones flecha, veamos una característica pequeña pero útil.

Digamos que tenemos el siguiente código:

function addOne(number) {

  return number + 1;

};

  

console.log(addOne(2)); // 3

console.log(addOne(5)); // 6

console.log(addOne()); // ¿Qué valor devuelve?

Prueba

¿Qué crees que pasará si llamas a addOne() sin ningún argumento? En algunos lenguajes de programación esto fallaría. Pero, en JavaScript, no fallará y tu código continuará ejecutándose.

El parámetro number recibirá un valor de undefined, por lo que la función devolverá undefined + 1, lo que resulta en NaN. Vimos en un capítulo anterior que podemos prevenir esto agregando una condición if.

Sin embargo, hay una manera más fácil. Echemos un vistazo a los parámetros por defecto.

Nota: Si estás confundido acerca de la diferencia entre parámetros y argumentos, aquí hay una breve definición: Un parámetro es una variable en la definición de una función. Cuando se llama a una función, los argumentos son los datos que pasas a los parámetros del método.
