Más métodos de array

En esta lección, veremos tres métodos de array.

Array map

El método .map(callback) te permite transformar un array en otro. Aquí hay algunos ejemplos comunes:

  • [4, 2, 5, 8] transformado a [8, 4, 10, 16]. Se duplicó cada elemento del array original.
  • ["sam", "Alex"] transformado a ["SAM", "ALEX"]. Se convirtieron todos los elementos del array original a mayúsculas.

Ten en cuenta que siempre obtienes un array que contiene la misma cantidad de elementos en comparación con el array original, pero cada elemento probablemente ha sufrido alguna transformación.

En el primer ejemplo, la transformación consiste en multiplicar cada número por 2.

En el segundo ejemplo, la transformación es llamar a .toUpperCase() en cada elemento.

Veamos cómo podemos implementar estas transformaciones:

const numbers = [4, 2, 5, 8];

const doubled = numbers.map(function(number) {

Si te olvidas de usar return dentro de la función de devolución, terminarás con el siguiente array: [undefined, undefined]. Esto se debe a que, por cada elemento en el array original (["sam", "Alex"]), estás devolviendo undefined, por lo que el resultado final será [undefined, undefined]. Array includes(item)

El método .includes(item) del array es uno de los métodos de array más simples, ya que toma un item en lugar de una función de devolución y devuelve true cuando ese item existe en el array y false de lo contrario. Aquí hay un ejemplo:

const groceries = ["Apple", "Peach", "Tomato"];

console.log(groceries.includes("Tomato")); // true

console.log(groceries.includes("Bread)); // false

Array.includes() en MDN.

Array join(glue)

Cuando tienes un array y lo renderizas en una página web (como veremos más adelante en la sección DOM del curso), el array se convertirá automáticamente en una cadena de texto. JavaScript invocará automáticamente el método .toString() del array, que devuelve una cadena de los elementos del array separados por comas. Así es como funciona:

const groceries = ["Apple", "Peach", "Tomato"];

console.log(groceries.join("; ")); // "Apple; Peach; Tomato"

console.log(groceries.join(". ")); // "Apple . Peach . Tomato"

Array.join() en MDN.

Resumen

  • El método .map(callback) del array te permite transformar un array en otro.
  • El método .includes(item) del array toma un item y devuelve true cuando ese item existe en el array y false de lo contrario.
  • El método .join(glue) del array devuelve una cadena de los elementos del array separados por el glue.

¿Se usa la aplicación?

Completa la función isAppUsed para que devuelva true cuando el parámetro app que recibe exista en el parámetro apps, y false de lo contrario.

function isAppUsed(apps, app) {

  return apps.includes(app);

}

 

// Ejemplo

console.log(isAppUsed( ['Twitter', 'Calculator'], 'Calculator')); // true

console.log(isAppUsed( ['Clock', 'Calculator'], 'Safari')); // false

Tamaño de la cadena

Completa la función getStringSizes para que devuelva un array con la cantidad de caracteres de cada cadena que recibe en el parámetro strings.

Es decir, para el array ["abc", "d"] debería devolver [3, 1], ya que la primera cadena tiene 3 caracteres y la segunda tiene 1.

function getStringSizes(strings) {

  return strings.map(function(string) {

    return string.length;

  });

}

 

// Ejemplo

console.log(getStringSizes( ['a', 'abc'])); // [1, 3]

console.log(getStringSizes( ['Sam', 'Alex', 'Charlie'])); // [3, 4, 7]

console.log(getStringSizes( ['Hello', 'Blue'])); // [5, 4]

Proyecto Aula I

¡Este es un proyecto de clase! Puedes ingresar calificaciones en la pestaña del navegador y agregarlas a la aplicación, que te mostrará varias estadísticas sobre el aula. Por ejemplo, verás el promedio de calificaciones, todas las calificaciones reprobadas, etc.

Cada función tiene un comentario que describe lo que debería devolver, pero también puedes encontrarlos a continuación:

  1. getNumberOfGrades debería devolver la cantidad de calificaciones.
  2. getSumGrades debería devolver la suma de todas las calificaciones.
  3. getAverageValue debería devolver el valor promedio de todas las calificaciones (suma de todas las calificaciones dividida por el número total de calificaciones).
  4. getPassingGrades debería devolver todas las calificaciones aprobatorias (10 o más).
  5. getFailingGrades debería devolver todas las calificaciones reprobatorias (9 o menos).
  6. getRaisedGrades debería devolver todas las calificaciones aumentadas en 1 (las calificaciones no deberían superar 20).

export function getNumberOfGrades(grades) {

  // TODO: devolver la cantidad de notas

  return grades.length;

}

 

export function getSumGrades(grades) {

  // TODO: devolver la suma de todas las notas

  let sum = 0;

  grades.forEach(function(grade) {

    return sum += grade;

  });

  return sum;

}

 

export function getAverageValue(grades) {

  // TODO: devolver el promedio de todas las notas

  let sum = 0;

  grades.forEach(function(grade) {

    return sum += grade;

  });

  return sum / grades.length;

}

 

export function getPassingGrades(grades) {

  // TODO: devolver todas las notas aprobatorias

  return grades.filter(function(grade) {

    return grade >= 10;

  });

}

 

export function getFailingGrades(grades) {

  // TODO: devolver todas las notas reprobatorias

  return grades.filter(function(grade) {

    return grade <= 9;

  });

}

 

export function getRaisedGrades(grades) {

  // TODO: devolver todas las notas aumentadas en 1, ninguna de ellas puede superar 20

  return grades.map(function(grade) {

    if(grade >= 20) {

      return 20;

    } else {

      return grade += 1;

    }

  });

}

export function getVotersCount(ages) {
