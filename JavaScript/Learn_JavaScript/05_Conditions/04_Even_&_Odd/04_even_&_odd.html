Pares e impares

En el capítulo de Números, explicamos el operador de resto (%). El resto de la división de un número entre dos es siempre 0 o 1. Esto te ayudará a saber si el número original es par (divisible por dos sin resto) o impar.

Veamos algunos números:

// números pares

4 % 2 // 0

6 % 2 // 0

8 % 2 // 0

10 % 2 // 0

 

// números impares

3 % 2 // 1

5 % 2 // 1

7 % 2 // 1

9 % 2 // 1

Observa cómo el resto de la división por dos es siempre 0 cuando el número es par. Por el contrario, es 1 cuando el número es impar.

En el próximo desafío, usaremos una condición if para mostrar si el número ingresado por el usuario es par o impar.

Resumen

  • El resto de la división de un número entre dos es siempre 0 o 1. Esto nos ayuda a determinar si el número es par o impar.

Prácticas

Hallar el resto de la división entre 2

Completa la función evenOrOdd para que devuelva la cadena "even" cuando el parámetro number que recibe es par y "odd" en caso contrario.

¿Puedes hacer que funcione también con números negativos?

export function evenOrOdd(number) {

  if (number % 2 === 0) {

    return "even";

  }

  return "odd";

}
