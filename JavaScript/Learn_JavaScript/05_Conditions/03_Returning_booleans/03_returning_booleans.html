Devolver valores booleanos

Cuando devuelvas un valor booleano (verdadero o falso), es bastante redundante usar if y else. Aquí tienes un ejemplo:

function isPassing(grade) {

  if (grade >= 10) {

    return true;

  } else {

    return false;

  }

}

 

console.log(isPassing(12));

Prueba

Esto es redundante porque grade >= 10 por sí solo se evalúa como true o false dependiendo de la nota. Esto significa que no necesitas envolverlo en una declaración if/else.

function isPassing(grade) {

  return (grade >= 10);

}

 

console.log(isPassing(12));

Prueba

sin usar if/else, lo cual siempre devolverá un valor booleano.

Esto solo funciona cuando estás devolviendo un valor booleano desde una función.

Resumen

  • Cuando devuelvas un valor booleano desde una función, puedes simplificar tu código devolviendo directamente la expresión booleana, sin usar una declaración if.

Prácticas

¿Puedes votar? (mejorada)

Implementa la función canVote para que devuelva true cuando la edad sea 18 o superior y false en todos los demás casos. No debes usar una condición if (o ternaria).

function canVote(age) {

  return (age >= 18);

}

 

console.log(canVote(25)); // true

console.log(canVote(18)); // true

console.log(canVote(17)); // false
